API Project: Timestamp Microservice

Usage :

  1. Visit api endpoint at localhost:portnumber
  2. Visit ...endpoint/api/timestamp for the current time
  3. Add a valid timestamp format after the /timestamp to get the time displayed as utc and standard (see example usage at the bottom)
  4. Service returns invalid error in json format if incorrect date/time is input in the url

Example Usage:

  • https://fcc-backend-projects.glitch.me/api/timestamp/2015-12-15
  • https://fcc-backend-projects.glitch.me/api/timestamp/1450137600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
